# ecommerce-website
A e-commerce website built using **HTML, CSS, and JavaScript**.  
This project demonstrates the fundamentals of frontend web development with a focus on layout, styling, and basic interactivity.

# Project Overview

This e-commerce website allows users to browse products through a clean and user-friendly interface.  
It is designed to be fully responsive and works smoothly across different screen sizes.

#  Features

- Responsive design for desktop and mobile devices  
- Product listing layout  
- Simple and clean user interface  
- Basic interactivity using JavaScript  
- Easy navigation  


# Technologies Used

- HTML – For structuring the web pages  
- CSS – For styling and responsive layout  
- JavaScript – For basic functionality and interactivity
